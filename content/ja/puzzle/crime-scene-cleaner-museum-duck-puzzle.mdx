## 美術館のアヒルパズルを理解する

モダンアート・ミッションのサイドギャラリーの一つにあるアヒルのインスタレーションは、インタラクトすると異なる音階を鳴らす一列のラバーダックで構成されています。小さなアヒルたちの横には、一回り大きな「マスター」ダックが鎮座しています。このため、多くのプレイヤーは隠し通路や実績を解除するために特定のメロディが必要だと信じてきました。

しかし、2026年時点のハイレベルなプレイテストとコミュニティのデータに基づくと、この**美術館のアヒルパズル**は、追跡対象のシークレットではなく、主にインタラクティブな「イースターエッグ（お遊び要素）」または音楽のおもちゃとして機能しているようです。アヒルは標準的なピアノのキーボードのように配置されており、ド（C）からシ（B）までの音に対応しています。配列の中でアヒルが「欠けている」箇所は、音楽理論におけるミ/ファ（E/F）とシ/ド（B/C）の間に半音（黒鍵）が存在しないことに対応しています。

| 特徴 | 説明 |
| :--- | :--- |
| **場所** | 「レッドルーム」入口近くのサイドパネル。 |
| **仕組み** | インタラクトによってトリガーされる音階（C-D-E-F-G-A-B）。 |
| **追跡対象のシークレット？** | **いいえ**。シークレットのカウンターには含まれません。 |
| **目的** | 演出・環境インタラクション。 |

> **💡 ヒント:** アヒルで遊ぶのは楽しいですが、「正解」のメロディを探して何時間も費やさないようにしましょう。ミッションを100%クリアしたい場合は、代わりに「TVパズル」や「カセットのジャンプパズル」に集中してください。

## TV迷路パズルの解き方（シークレット #2）

シークレットとしてカウントされる実際の「パズル」を探しているなら、様々なテレビ画面が設置された迷路のようなインスタレーションに向かう必要があります。ここは環境的なヒントを見逃しやすいため、多くのプレイヤーが詰まるポイントです。

以下の手順で隠し扉をアンロックしてください。

1.  **ヒントを確認する:** 迷路に入る前に、部屋の天井を見上げてください。頭上のモニターに特定の色パターンが表示されているのが見えます。
2.  **迷路を進む:** 迷路に入り、壊れて砂嵐が流れているテレビが集まっている左奥のコーナーへ向かいます。
3.  **色を合わせる:** テレビを調べて砂嵐の色を変更します。天井のモニターに表示されていたシーケンスと全く同じ色に合わせる必要があります。
4.  **隠し部屋に入る:** 色が正しく揃うと、右側の隠し扉がスライドして開きます。

この部屋の中には「刀（Katana）」があります。この武器を拾うと、マネキンとの特殊な戦闘イベントが発生します。防衛に成功すると出口が開き、外にカセットテープを持った新しいマネキンが出現します。これにより実績**「Rage Against the Machine」**も解除されます。

## モダンアートの全カセットテープの場所

美術館内には4つのカセットテープが隠されています。これらを集めることは、[Crime Scene Cleaner サウンドトラック](https://store.steampowered.com/app/1040200/Crime_Scene_Cleaner/)を楽しみながら作業したいプレイヤーにとって必須です。

| テープ番号 | 場所 | 入手方法 |
| :--- | :--- | :--- |
| **1** | レッドルーム入口 | 赤い扉を入ってすぐ、火災報知器の近くのテーブルの上。 |
| **2** | コンサートステージ | ステージ上のマネキンを修理する。照明が暗くなり音楽が流れると、床にテープが出現する。 |
| **3** | 2階の廊下 | アヒルの部屋の近く。バリアを飛び越えて階段を上がった先の、木箱近くの床にある。 |
| **4** | 刀の部屋 | 刀のイベント完了後、隠しTV部屋の外に出現するマネキンが持っている。 |

## カセットの壁ジャンプシークレット（シークレット #1）

このミッションの最初の公式シークレットは、コンサートステージ（カセットテープ #2を見つけた場所）と同じ部屋にあります。これは正確な操作が求められるアスレチックチャレンジです。

1.  **壁を探す:** 大きな透明のカセットテープの模型で飾られた壁を探します。
2.  **足場を起動する:** これらの装飾はインタラクト可能です。クリックすると壁からスライドして飛び出し、一時的な足場になります。
3.  **登る:** 一番下のカセットから始めます。クリックして上に飛び乗り、さらに高い位置にある次のカセットをクリックします。一定時間で引っ込んでしまうため、素早く動く必要があります。
4.  **隠しスイッチ:** 壁の最上部に、小さなカセットの装飾があります。この小さなスイッチを調べると、壁の高い位置にある隠し扉が開きます。
5.  **報酬:** 開口部に飛び込むと、隠された現金とレアな絵画が見つかります。これがミッションの公式シークレット1つ目としてカウントされます。

## モダンアートでの必須掃除テクニック

このマップは非常に広大であるため、効率が重要です。4時間かけて掃除した後に、像の裏に血痕を見落としていたことに気づくのは避けたいものです。

*   **モップのアップグレードを使用:** 「レッドルーム」の圧倒的な血の量を処理するために、高容量のモップは必須です。
*   **死体の処理:** 本格的な清掃を始める前に、すべての死体を袋に入れて回収地点まで運んでおきましょう。後で移動させると、せっかく綺麗にした場所を汚してしまう可能性があります。
*   **天井をチェック:** モダンアート・ミッションは天井が高く、吊り下げられた展示物が多いため、爆発などによる返り血がそれらに付着していることがよくあります。
*   **持ち物の管理:** バケツを近くに置いておきましょう。この美術館では、水源まで戻る距離がこれまでのミッションよりも大幅に長くなっています。

> **警告:** 壊れやすい美術品の近くで高圧洗浄機を使用する際は注意してください。美術館の所有物を破損させると、ミッション終了時に多額の罰金が課せられる可能性があります。

## FAQ

**Q: 美術館のアヒルパズルで実績は解除されますか？**

A: いいえ、アヒルを演奏しても特定の実績は解除されませんし、シークレットの合計数にもカウントされません。このレベルの主な実績は、刀を見つけることと、高い評価で清掃を完了することに関連しています。

**Q: TVの色を合わせましたが、ドアが開きません。何が問題でしょうか？**

A: 天井のヒントをもう一度確認してください。地面からの視点だと、2つの色を入れ違えて見てしまうことがよくあります。また、クラスター内のすべてのテレビとインタラクトして、シーケンスを「確定」させているか確認してください。

**Q: 刀はどこにありますか？**

A: 刀はTVの色パズルでアンロックされる隠し部屋の中にあります。これを拾うことが、マネキン戦の発生と4本目のカセットテープ入手の条件となります。

**Q: モダンアート・ミッションには公式にいくつのシークレットがありますか？**

A: 公式に追跡されるシークレットは2つです。「カセット壁のジャンプパズル」と「TV迷路の色パズル」です。**美術館のアヒルパズル**は注目を集めやすい要素ですが、公式な2つのシークレットには含まれません。
