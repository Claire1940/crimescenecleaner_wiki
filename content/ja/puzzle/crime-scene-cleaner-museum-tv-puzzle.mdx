## モダンアート・ミッションの概要

## 美術館のテレビパズルの解き方

**美術館のテレビパズル**は、テレビ展示棟にあります。まず、巨大なティラノサウルスの骨格があるメインホールからスタートします。ティラノサウルスから左へ進み、右手にある緑色のドアを通り抜けます。すると、モニターが並ぶ迷路のようなセクションにたどり着きます。

### ステップ 1: 色のヒントを見つける
パズルの答えはランダムではありませんが、環境の中にひっそりと隠されています。必要な色を特定する方法は2つあります。
1.  **天井の反射:** 壊れたテレビがある部屋で、真上の天井を見上げてください。特定の照明条件下では、正しい色のシーケンスが反射しているか、頭上のモニターに表示されています。
2.  **絨毯展示室の奥の部屋:** 天井が暗すぎて読めない場合は、絨毯の展示エリアへ向かってください。奥の部屋（「仕事に来るな」というメモがある部屋）に、パズルに必要な正確な色の順序を表示している稼働中のモニターセットがあります。

### ステップ 2: シーケンスを入力する
色がわかったら、テレビ展示エリアの左奥の角に戻ります。砂嵐が流れる壊れたモニターの集まりがあります。これらの画面を操作して、ヒントの色と一致するまで色を切り替えてください。

| モニターの位置 | 必要なアクション | 結果 |
| :--- | :--- | :--- |
| **左の画面** | ヒントの色に合わせる | 砂嵐の色が変わる |
| **中央の画面** | ヒントの色に合わせる | 砂嵐の色が変わる |
| **右の画面** | ヒントの色に合わせる | **隠し扉が開く** |

> **⚠️ 重要な注意:** もしテレビ画面が単色のグレーで表示され、色や砂嵐が見えない場合は、グラフィック設定の競合が発生している可能性があります。ビデオ設定で **アンビエントオクルージョン (Ambient Occlusion)** がオンになっていることを確認してください。2026年現在のビルドでは、これが有効になっていないとパズルの色が正しくレンダリングされないことがあります。

## 刀と秘密のエリア #2 の解放

**美術館のテレビパズル**を正常に完了すると、モニターの右側にある隠し扉がスライドして開きます。中には、美術館の他の場所とは明らかに雰囲気が異なる、薄暗い部屋があります。

この部屋の突き当たりに「刀」が置かれています。これを拾うと戦闘イベントが発生します。マネキンが出現して襲ってくるので、刀を使って身を守ってください。すべての敵を「清掃」すると出口が開き、外に新しいマネキンが現れて **カセットテープ #4** を持っています。この一連の流れにより、実績「Rage Against the Machine」も獲得できます。

## モダンアートのすべてのカセットテープの場所

テレビパズルが最も複雑ですが、他の収集品を見つけるにも鋭い観察眼と多少のアスレチック要素が必要になります。

### カセットテープ #1: 赤い部屋
これは最も簡単に見つかるテープです。ミッション開始直後、右手にある赤いドアを探してください。部屋に入り、火災報知器の近くのテーブルを見ると、そこにテープが置いてあります。

### カセットテープ #2: コンサートステージ
このテープは、ちょっとした環境ギミックと連動しています。
1.  ガラス越しのコンサートステージがある部屋に入ります。
2.  横のドアから展示の裏側に回ります。
3.  ステージ上の倒れたマネキンを直し、死体があれば片付けます。
4.  「セット」が復元されると、ライトが点滅して音楽が流れ始めます。テープがステージの床に出現します。

### カセットテープ #3: 2階のアヒル部屋
スタート地点から、アヒルの彫刻でいっぱいの部屋に向かいます。右側にバリケードで塞がれた階段があります。バリケードを飛び越えて上階へ進みます。廊下を右に進むと、収納ボックスの横の床にテープが落ちています。

## 秘密のエリア #1: カセットの壁

ミッションの最初の秘密（テレビパズルとは別）は、コンサートステージと同じエリアにあります。壁の一つに、大きな透明なカセットの装飾があることに気づくでしょう。これらは単なる飾りではなく、インタラクティブな足場になっています。

| ステップ | アクション |
| :--- | :--- |
| **1** | 一番低い位置にある壁のカセットを操作して引き出す。 |
| **2** | 引き出されたカセットの上に飛び乗る。 |
| **3** | 上にあるカセットを順次操作して「階段」を作る。 |
| **4** | 一番上で、隠された小さなカセットスイッチを探す。 |
| **5** | それを起動すると壁のパネルが開き、現金とレアな絵画でいっぱいの部屋に入れます。 |

## 2026年版テクニカル・トラブルシューティング

『Crime Scene Cleaner』には様々なアップデートが行われてきたため、新しいハードウェアを使用しているプレイヤーは美術館の照明やパズルで問題が発生することがあります。

> **💡 ヒント:** 美術館内が暗すぎて進めない場合は、「センス (Sense)」能力を頻繁に使用してください。テクスチャが正しく読み込まれていない場合でも、パズルのテレビ画面を含むインタラクティブなオブジェクトがハイライトされます。

| 問題 | 解決策 |
| :--- | :--- |
| **テレビが単色のグレーになる** | グラフィック設定で **アンビエントオクルージョン** を有効にする。 |
| **刀が出現しない** | パズルのすぐ近くにある血痕をすべて清掃したか確認する。 |
| **マネキンが倒れない** | 強攻撃（右クリック）を使用して、彼らのモーションを中断させる。 |
| **カセット #4 が見つからない** | 隠し部屋にいるすべてのマネキンを倒さないと、外に出現しません。 |

さらに詳しいコミュニティ発の修正方法は、ゲームの [公式Steamディスカッションページ](https://steamcommunity.com/app/1040200/discussions/) で確認できます。

## よくある質問 (FAQ)

**Q: 部屋を掃除する前に美術館のテレビパズルを解くことはできますか？**

A: はい、部屋に入ってすぐにパズルを解くことが可能です。ただし、色を合わせている最中に滑ったり足跡をつけたりしないよう、先に床のガラスや血痕を片付けておく方が安全です。

**Q: 刀をなくしてしまったらどうなりますか？**

A: 刀は隠し部屋の戦闘イベント用の「一時的な」武器です。永続的な清掃ツールとして持ち歩くことはできませんが、一度拾えばプロフィールに関連する実績が永久にアンロックされます。

**Q: 色を見つけたのにドアが開きません。何が間違っているのでしょうか？**

A: 順番を再確認してください。シーケンスは、ヒント（天井または絨毯展示室の奥）に表示されている通り、左から右に読み取る必要があります。それでもダメな場合は、各テレビの色を一周させてインタラクションを「リセット」してみてください。

**Q: モダンアート・ミッションは最終レベルですか？**

A: いいえ、モダンアートは最後から2番目のミッションです。ここで秘密とカセットを100%回収しておくことで、ゲームの最終清掃チャレンジに向けた完璧な準備が整います。
