## 主要なスコアリングカテゴリを理解する

Crime Scene Cleanerのスコアリングシステムは、いくつかの異なる指標に基づいてパフォーマンスを評価します。主な目標は明らかに犯罪の痕跡をすべて消し去ることですが、ゲームは基本的な要件を超えて行動するプレイヤーに報酬を与えます。

| カテゴリ | 最終評価への影響 | 説明 |
| :--- | :--- | :--- |
| **清掃率** | 高 | 現場から除去された血痕、汚れ、足跡の総量。 |
| **ゴミの処分** | 中 | すべてのゴミ、割れたガラス、廃棄されたアイテムを袋やビンに回収すること。 |
| **証拠/手がかり** | 高 | 犯罪の経緯に関連する特定のアイテムを見つけること。 |
| **家具の復元** | 低 | 動かされたり倒されたりした家具を、元の場所や論理的な位置に戻すこと。 |
| **死体の処理** | 極めて重要 | ミッションのパラメータに従って、すべての遺体が除去または処理されていることを確認すること。 |

## Crime Scene Cleaner スコアリングシステムガイドの仕組み

Sランクを達成するには、スピードと細部への細心の注意のバランスをとる必要があります。**Crime Scene Cleaner スコアリングシステムガイド**では、一部のボーナス目標において時間は要因となりますが、評価に最も大きく貢献するのは徹底さであると強調しています。ほとんどのミッションでは合格に少なくとも95%の清掃しきい値が必要ですが、完璧主義者は100%とすべての収集品を目指すべきでしょう。

### 清掃率
ゲームは血の一滴一滴まで追跡します。広い範囲には高圧洗浄機を、デリケートな表面にはモップを使用することが不可欠です。スコアが99%で止まってしまった場合は、天井やテーブルの裏を確認してください。足跡も、見落としがちな汚れの代表格です。血の中を歩いた後にきれいな床の上を歩くと、自分で余計な仕事を作ることになります。

### 証拠と物語の手がかり
手がかりは単なる演出ではありません。最終スコアにとって不可欠な要素です。各ミッションには、現場で何が起こったのかを説明する、電話、メモ、デジタルデバイスなどの一定数の手がかりが設定されています。たとえ部屋がピカピカであっても、手がかりを一つでも逃すと評価が大幅に下がる可能性があります。

## ミッションスポットライト：「Call In Dead」攻略戦略

最終ミッション「Call In Dead」は、ゲーム内で最も複雑なチャレンジの一つです。複数のフロア、コンベア、クレーンがある広大な工業地帯が舞台となります。ここで完璧なスコアを獲得するには、一連の電力パズルとキーカードの要件をクリアする必要があります。

### キーカードの場所と電力管理
「Call In Dead」では、すべてのシステムを一度にアクティブにすることはできません。アクティブにできるパネルは3つに制限されています。手がかりや秘密が隠されているエリアにアクセスするためには、電力供給の管理が重要です。

| キーカードの色 | 場所 | 機能 |
| :--- | :--- | :--- |
| **緑** | 1階、コンベアベルトの上。 | 基本的なコンベアシステムを起動する。 |
| **青** | 1階、ミニゲームのドアの奥。 | ハッチの制御とアクセスに使用。 |
| **オレンジ** | 1階の「男（NPC）」の近く。 | エレベーターの起動に不可欠。 |
| **黄** | 2階、エレベーター経由で入手。 | コンテナ移動用のクレーンを操作する。 |

### 4つの不可欠な手がかりを見つける
「Call In Dead」では、**Crime Scene Cleaner スコアリングシステムガイド**の物語完了の要件を満たすために、4つの手がかりを収集する必要があります。

1.  **電話 1:** メインフロアにあり、開始地点の近くで簡単に見つかります。
2.  **電話 2:** 2階にあります。エレベーターに電力を供給する必要があります（オレンジのキーカード）。
3.  **ボード:** クレーンエリアにあります。コンテナを飛び移る必要があります。
4.  **コンテナの手がかり:** クレーンで下ろす必要がある特定のコンテナの中にあります。

## 秘密とカセット：コンプリートへの道

厳密には「清掃」スコアの一部ではありませんが、秘密やカセットを見つけることで、プロフィール全体やアンロック可能なコンテンツが大幅に強化されます。「Call In Dead」ミッションでは、倉庫全体に5つの秘密と4つのカセットが隠されています。

### 秘密の部屋の場所
*   **秘密 1:** 入口近くの通気口やパイプをジャンプして進むことで、開始直後にアクセス可能です。
*   **秘密 2:** 電源近くで見つかる「Room Key（部屋の鍵）」が必要です。これにより、最初のカセットが入っている隠しオフィスが開きます。
*   **秘密 3:** 2階にあり、金庫にたどり着くためのプラットフォームアクションが必要です。障壁を突破するために刀や斧などの道具が必要になる場合があります。
*   **秘密 4:** 「クレイジーなブレンド男」に会う電源エリア近くの「隠し壁」の背後にあります。
*   **秘密 5:** 「Unknown Room（未知の部屋）」。クレーンでアクセス可能なコンテナの中にある「Unknown Key（未知の鍵）」と、特定の電力設定（ハッチ電源オン）が必要です。

### カセットチェックリスト
1.  **カセット 1:** 秘密の部屋2の中にあります。
2.  **カセット 2:** 2階のメインオフィスエリアにあります。
3.  **カセット 3:** クレーンベイの左側のコンテナに隠されています。
4.  **カセット 4:** 最後の秘密の部屋（秘密の部屋5）の中にあります。

## Sランク獲得のための高度なヒント

**Crime Scene Cleaner スコアリングシステムガイド**で一貫して高い評価を得るには、ミッションを開始する前に装備を最適化する必要があります。モップの吸収力や高圧洗浄機の射程をアップグレードすることで、大幅な時間を節約できます。

*   **ゴミを忘れない:** 置き去りにされたゴミはすべて減点対象です。ゴミ袋を効率的に使い、指定された処分箱やトラックに確実に捨ててください。
*   **略奪 vs プロ意識:** 現場で現金や貴重品を見つけることができますが、過度な略奪は「プロ意識」の評価に影響を与えることがある点に注意してください。ただし、資金繰りには役立ちます。
*   **「清掃なし」ラン:** 「Call In Dead」などの特定のミッションでは、清掃を行う前にすべての手がかりと秘密を収集することが可能です。これは、本格的な清掃に取り組む前にマップを偵察するのに適した方法です。

> **警告:** 重い物を動かすときは注意してください。汚れた水の入ったバケツをひっくり返すと、ひどい汚れが広がり、数分間の進捗が瞬時に台無しになり、効率スコアが激減します。

## FAQ

**Q: ミッション完了までにかかる時間は、Crime Scene Cleanerのスコアリングシステムに影響しますか？**

A: 2026年の現行バージョンでは、時間はベーススコアに直接的なペナルティを与えませんが、ミッションを素早く完了させることで「スピード」ボーナスを獲得し、グローバルリーダーボードで上位に入ることができます。

**Q: ミッションで手がかりを一つでも逃すとどうなりますか？**

A: 手がかりを逃すと、通常はSランクを達成できなくなります。ほとんどのミッションでは、最高評価を得るために100%の手がかり発見が必要です。

**Q: 主要な目的を終えた後、エリアに戻って清掃することはできますか？**

A: はい、車両を操作して「現場を離れる」までは、清掃を続けたり秘密を探したりすることができます。出発前にUVライトで最終チェックを行うことを強くお勧めします。

**Q: 清掃率100%を達成するためにカセットは必要ですか？**

A: いいえ、カセットと秘密は「収集品」として扱われます。これらは全体的な達成率には貢献しますが、清掃や手がかり発見のスコアとは別に追跡されます。ただし、「完璧な」プレイを目指すなら、ほとんどのプレイヤーはすべてを収集することを目指します。
