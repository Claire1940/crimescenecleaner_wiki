## 重要なシークレットの仕組み

レベル別のガイドに入る前に、2026年版のゲームにおけるシークレットの分類を理解しておくことが重要です。

*   **シークレットの場所（Secret Locations）：** 通常、多額の現金や「カタナ」のようなユニークなアイテムが入っている隠し部屋や区画です。
*   **カセットテープ（Cassette Tapes）：** ノーマルモードとナイトメアモードの両方で見つかり、ミュージックプレーヤーの新しいトラックをアンロックします。
*   **ミュージックディスク（Music Disks / ナイトメアモード）：** ナイトメアモード限定の要素で、特定のパズルを解く必要があり、通常のシークレットに代わって登場します。
*   **証拠品（Evidence）：** 厳密には「シークレット」ではありませんが、一部の証拠品は隠し扉の奥にあり、パーフェクトスコアを達成するために必要です。

> **💡 ヒント：** 「クリーナーセンス」（デフォルトはVキー）を頻繁に使用しましょう。緩んだ床板や傾いた壁掛け照明など、シークレットエリアにつながるインタラクティブなオブジェクトを強調表示してくれます。

## Bad Call：シークレットと収集アイテム

最初のミッションでは、ゲームのシークレットにおける垂直性の概念が導入されます。アパートの内部と外壁の縁（レッジ）の両方を探索する必要があります。

| シークレットの種類 | 場所 | 獲得条件 |
| :--- | :--- | :--- |
| **シークレット1：植物の部屋** | 鉄の扉の奥にある中庭の奥 | 2つの鍵を見つける（1つは箱の上、1つはバスルーム） |
| **シークレット2：リフト下の部屋** | ディーラーのアパートの真下の階 | バルコニーから外壁の縁を伝って歩く |
| **シークレット3：屋上の隠れ家** | 隣の建物の屋上 | 廊下の窓を塞いでいる木板を壊す |
| **カセット：Cieszyn's Venice** | 植物の部屋の内部 | シークレットルーム内にある2番目の扉をアンロックする |

### ナイトメアモード：ポラロイドの探索
ナイトメアモードでは、ミュージックディスクは暗室の中にロックされています。屋根裏部屋で**ポラロイドカメラ**を見つけ、以下の4つの特定の写真を撮る必要があります。
1.  **屋根裏部屋：** 奥の部屋の草が生い茂っている裏側。
2.  **縁（レッジ）：** ノーマルモードのシークレットがある廊下の窓の外。
3.  **庭：** ゴミ捨て場の近く。
4.  **下の階のアパート：** 木のスラットを壊して侵入した先。

## Trial By Blood：スパのシークレット

このレベルは、キーカードの連鎖に大きく依存しています。ここで**Crime Scene Cleanerのすべてのシークレット**を見つけるには、計画的に探索する必要があります。

1.  **ロッカー M034：** 日焼け室のテーブルの上にあるキーカードを見つけます。
2.  **VIPカード：** 男子ロッカールームのロッカーM034を開け、中にあるVIPカードを入手します。
3.  **VIPルーム：** デジタルロック室（ロッカールームの向かい）にあるコントローラーにVIPカードを使用します。これにより、プールエリアの隠し壁が開きます。
4.  **ロッカー W018：** VIPルームのコーヒーテーブルからW018カードを取り、女子トイレで使用します。

> **⚠️ 警告：** デジタルロック室ではハッキングのミニゲームが必要です。動く線がハイライトされた数字に重なったときに、タイミングよくクリックしてください。

## Toxic Love：隠された絵画

このミッションは比較的コンパクトですが、シークレットはヴィラの装飾の裏に巧みに隠されています。

*   **メインシークレット：** リビングルームで、2つのランプの間に飾られた縦長の絵画を探してください。それを調べると、隠し区画が現れます。
*   **サブスイッチ：** 最初の隠し区画の中に、ヒューズボックスの近くにもう一つの絵画があります。ここのスイッチを入れると、1階と2階の両方の扉が開きます。
*   **カセット：Purrfect Grooves：** メインリビングエリアの棚にあります。

## Short Circuit：傾いた壁掛け照明

Short Circuitには、環境への干渉を伴う、最も「クラシック」なビデオゲームらしいシークレットが登場します。

| シークレットの場所 | アクセス方法 |
| :--- | :--- |
| **傾いた照明（Sconce）** | 2階（日本式1階）廊下の左側にある2番目の壁掛け照明を調べる。 |
| **冷蔵庫の貯蔵庫** | キッチンにある冷蔵庫の隣の、非常に目立たない小さな扉を探す。 |
| **宇宙飛行士の箱** | （ナイトメアモード）オフィスにある割れた窓から飛び出し、外壁の縁に到達する。 |

## Italian Job：ピザ屋とカジノ

Italian Jobで**Crime Scene Cleanerのすべてのシークレット**を見つけるには、文字通り「表面下」を見る必要があります。

*   **賭博場：** ピザ屋の入り口の左側にある部屋へ行きます。ダーツボードの下にある箱をどかすと、トラップドアが見つかります。鍵はキッチンの引き出しの中にあります。
*   **秘密のコンサート：** 裏路地にレンガの壁があります。近くにあるか、スキルツリーで取得したハシゴを使って壁を乗り越えてください。壁の裏には、亡くなったウェイトレスとカセットテープがあるステージがあります。
*   **カセット：Swishi Swoop：** 隠し賭博場の中の引き出しの上にあります。

## Affair with Death：ヴィラのシークレット

このレベルは広い屋外エリアと多層階のヴィラが特徴です。多くのシークレットはバスケットボールコートや地下室に関連しています。

*   **ドラッグの角：** バスケットボールコートへ行き、階段を上って縁を角まで進みます。最も高い木の下に、薬物と現金の隠し場所があります。
*   **現金のトロリー：** 地下室で、積み上げられた箱の山を探してください。それらを動かすと、現金が詰まったトロリーがある隠し部屋への這い出し口が現れます。
*   **ランドリールーム：** 2階（日本式1階）のバスルームのトイレの上に鍵があります。それを使って1階（地上階）のランドリールームを開けます。

## Modern Art：美術館のパズル

Modern Artには、色合わせパズルと伝説の武器が絡む、ゲーム内で最も複雑なシークレットが含まれています。

### カタナのシークレット
1.  恐竜の入り口から迷路の部屋に入ります。
2.  ガラスの天井または監視モニターを見て、特定のカラーパターンを確認します。
3.  迷路の奥にある、複数のモニターが設置された場所へ進みます。
4.  モニターのチャンネルを切り替え、天井で確認したパターンに合わせます。
5.  壁が開き、**カタナ（日本刀）**が現れます。この武器を使ってマネキンを切り裂き、部屋から脱出する必要があります。

### ゴールデンカセット
コンサートホール（壁に透明なカセットケースが並んでいる部屋）では、ケースをハシゴのように登ることができます。一番上まで到達するとゴールデンカセットが見つかり、それを調べると近くの隠し扉が開きます。

## Call in Dead：工場のシークレット

最終メインミッションは工業地帯の迷宮です。ここで**Crime Scene Cleanerのすべてのシークレット**を見つけるには、慎重なプラットフォームアクション（足場移動）が必要です。

| シークレットルーム | 場所・方法 |
| :--- | :--- |
| **庭師のオフィス** | 電源供給室のスイッチの隣にある通気口を壊す。 |
| **テレビの部屋** | ホール1階で鍵を見つける。部屋は電源供給室のカーテンの裏にある。 |
| **開発者のオフィス** | コントロールルームでロッカーを登り、高い位置にある通気口を進む。 |
| **エジプトの部屋** | 「カーゴ（貨物）」エリアの青いコンテナの中で鍵を見つける。仕分け室の扉をアンロックする。 |

> **💡 ヒント：** カーゴエリアでは、クレーンを使って黄色いコンテナを持ち上げてください。これは最後の証拠品と隠された現金の山を見つけるために不可欠です。

## よくある質問（FAQ）

**Q: 100%のスコアを取るために、すべてのシークレットを見つける必要がありますか？**

A: はい。主な目的は清掃ですが、「見つけたシークレット（Secrets Found）」の指標は最終評価に影響し、コヴァルスキーの娘の医療費として受け取れるボーナス現金の額にも関わります。

**Q: ナイトメアモードではシークレットの内容が変わりますか？**

A: 場所は概ね同じですが、報酬が変わります。ナイトメアモードでは、ノーマルモードで見つかる通常の現金の代わりに、ミュージックディスクや不気味なオルゴールを探すことになります。

**Q: レベルをクリアした後で、シークレットを探しに戻ることはできますか？**

A: コヴァルスキーの家にあるメインコンピューターから、どのミッションでもリプレイ可能です。これは、初回のプレイで見逃した**Crime Scene Cleanerのすべてのシークレット**を回収するのに最適な方法です。

**Q: カタナはどこにありますか？**

A: カタナは「Modern Art」レベルにあります。ガラスの迷路の部屋でモニターの色パズルを解くと、刀が入った隠し区画が開きます。
