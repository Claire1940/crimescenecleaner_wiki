## Secluded Retreat : Les secrets de l'hôtel Peaks

La mission "Secluded Retreat" vous emmène dans les montagnes enneigées, où l'hôtel Peaks sert de façade à des activités véritablement dépravées. Ce niveau est célèbre pour ses systèmes de sécurité automatisés devenus incontrôlables, laissant derrière eux une traînée de débris humains et mécaniques.

### Trouver les CD de musique
Il y a plusieurs CD de musique cachés dans tout l'hôtel. Pour les trouver, vous devez explorer des pièces qui ne font pas nécessairement partie de votre parcours de nettoyage principal. Vérifiez les tables de chevet dans les suites des invités et les étagères dans les zones de maintenance réservées au personnel.

### La salle de sécurité et le bureau de Lars
Pour terminer le niveau parfaitement, vous devez accéder à la salle de sécurité pour effacer les enregistrements. Cette pièce est cachée derrière une porte verrouillée par un code près du hall principal. Une fois à l'intérieur, vous pouvez également trouver un registre secret qui lie les activités "kinky" de l'hôtel au sénateur Holden. Le bureau de Lars est situé à l'étage supérieur ; cherchez la fenêtre par laquelle il est censé s'être échappé pour trouver une réserve d'argent cachée sous les lattes du plancher.

## Amber Alert : Les secrets du cercle souterrain

"Amber Alert" est peut-être la mission la plus chargée d'émotion de l'acte 2. Elle révèle le lien entre la fille du mafieux et le réseau de trafic d'êtres humains dirigé par Dave. Le nettoyage ici est colossal, s'étendant sur une salle de boxe, un vestiaire et un sous-sol caché.

### Salle secrète : Le centre de trafic
Derrière les casiers du vestiaire des "perdants", se trouve un panneau amovible. L'actionner révèle un couloir caché menant au centre de trafic. Cette zone contient les éléments d'histoire les plus importants de l'acte, notamment une conversation enregistrée entre Dave et le sénateur Holden.

### Liste de contrôle des objets de collection : Amber Alert
Trouver tous les secrets de ce niveau nécessite de vérifier les poutres au-dessus du ring de boxe. Utilisez l'échelle près du stockage d'équipement pour atteindre les passerelles.

| Type d'objet | Emplacement | Description |
| :--- | :--- | :--- |
| **CD de musique n°4** | Poste du barman | Caché derrière le shaker sur l'étagère inférieure. |
| **Note secrète** | Bureau privé de Dave | Sous le bureau, détaillant l'offre spéciale "deux pour le prix d'un". |
| **Trophée Pantalon d'Or** | Salon VIP | Une référence à une chanson de rap célèbre, trouvée dans une vitrine verrouillée. |

## Guide complet des secrets de l'acte 2 de Crime Scene Cleaner : Conseils de pro

Pour obtenir une note parfaite dans l'acte 2, vous devez aller au-delà du simple nettoyage. Le jeu récompense la minutie et la capacité à trouver des objectifs "non répertoriés".

1.  **Vérifiez les conduits d'aération :** De nombreuses salles secrètes de l'acte 2 sont reliées par des conduits de ventilation. Si vous trouvez un tournevis, gardez-le sur vous : c'est la clé de plusieurs caches d'argent cachées.
2.  **Efficacité de l'élimination des corps :** Dans des niveaux comme "Secluded Retreat", utilisez les incinérateurs industriels ou le broyeur de végétaux situé dans le hangar de maintenance. Éliminer les corps de ces manières uniques déclenche souvent des succès cachés.
3.  **La trace du sénateur :** Gardez un œil sur les objets marqués du blason "Holden". Les collecter ne fait pas qu'augmenter votre score ; cela débloque la véritable fin du jeu en fournissant suffisamment de preuves contre le politicien corrompu.
4.  **Nettoyer avant de réparer :** Nettoyez toujours le sang autour d'une machine cassée avant d'essayer de la réparer. Si vous réparez la machine d'abord, les "étincelles" peuvent parfois masquer de petites gouttes de sang, rendant impossible l'obtention des 100 %.

## Mécaniques cachées dans l'Acte 2

L'acte 2 introduit la mécanique de "Falsification de preuves". Contrairement à l'acte 1, où vous vous contentiez de retirer des objets, vous devez ici parfois replacer des objets pour faire passer une scène pour un accident. Par exemple, dans l'hôtel Peaks, vous devrez peut-être déplacer le membre d'un robot pour faire croire à une défaillance mécanique plutôt qu'à un massacre.

| Mécanique secrète | Comment l'activer | Récompense |
| :--- | :--- | :--- |
| **L'aspect "Accident"** | Réorganiser les meubles pour couvrir les impacts de balles | Argent bonus + Suspicion réduite |
| **Amateur de pizza** | Trouver les 5 boîtes de pizza dans les égouts/Amber Alert | Succès "Teenage Mutant" |
| **Fantôme dans la machine** | Pirater l'ordinateur de Lars sans déclencher l'alarme | Entrée d'histoire secrète |

## FAQ

**Q : Quel est le secret le plus difficile à trouver dans ce guide des secrets de l'acte 2 de Crime Scene Cleaner ?**

R : La plupart des joueurs ont du mal avec le CD de musique de la mission "Secluded Retreat". Il est caché derrière un miroir cassable dans la suite nuptiale. Vous devez briser la glace pour trouver la petite alcôve contenant le disque.

**Q : Dois-je trouver tous les secrets pour débloquer l'acte suivant ?**

R : Bien que vous n'ayez pas besoin de 100 % des secrets pour progresser, trouver les preuves contre le sénateur Holden est nécessaire pour débloquer la voie "Justice" en fin de jeu. Sans ces secrets, vous pourriez être forcé vers la "Mauvaise" fin.

**Q : Puis-je revenir en arrière et trouver les secrets que j'ai manqués ?**

R : Oui, vous pouvez utiliser le menu de sélection de mission depuis votre ordinateur dans la zone centrale. Cela vous permet de rejouer n'importe quelle mission de l'acte 2 avec vos outils améliorés, ce qui facilite grandement la recherche des zones cachées manquées lors de votre premier passage en 2026.

**Q : Comment gérer les robots tueurs dans Secluded Retreat ?**

R : Vous ne pouvez pas "combattre" les robots. Vous devez utiliser la discrétion et l'outil de piratage trouvé dans le hall pour les désactiver. Une fois désactivés, ils deviennent des "déchets" que vous devez jeter dans les grandes bennes à l'extérieur.
