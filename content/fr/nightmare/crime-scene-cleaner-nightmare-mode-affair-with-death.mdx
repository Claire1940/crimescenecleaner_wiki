Dans cette mise à jour 2026, le passé de Kovalsky le hante dans une version surréaliste et ensanglantée de la Villa de montagne. Contrairement à la mission standard, la version Cauchemar déplace votre point de départ, brouille les énigmes et ajoute des couches de complexité à la narration environnementale.

Pour obtenir un score parfait dans **Crime Scene Cleaner Nightmare Mode Affair with Death**, vous devez localiser un seul corps insaisissable, collecter trois éléments de preuve critiques et résoudre une série d'énigmes complexes à pavé numérique disséminées dans le domaine. ## Aperçu de la mission et objectifs clés

Dans la version Cauchemar d' "Affair with Death", l'échelle de l'environnement reste au niveau 3, mais la nature "tordue" de la carte signifie que de nombreux raccourcis du jeu standard sont bloqués ou nécessitent de nouvelles clés. Votre objectif principal est de nettoyer les énormes taches de sang, d'éliminer le seul corps qui tombe du balcon et de remettre 30 meubles à leur place.

| Type d'objectif | Nombre total | Lieux principaux |
| :--- | :--- | :--- |
| **Corps à éliminer** | 1 | Tombe de la chambre du premier étage vers la piscine |
| **Éléments de preuve** | 3 | Chambre, Sous-sol, Salon |
| **Meubles à déplacer** | 30 | Hall, Cuisine, Salon, Chambres |
| **Objets secrets** | 1 | Disque de musique ambré (Zone de la piscine) |

> **💡 Conseil :** Commencez toujours par trouver les points d'eau. En mode Cauchemar, la salle de bain de départ au premier étage est votre source initiale, mais vous devrez déverrouiller l'allée et la piscine pour accéder à des stations de recharge plus pratiques.

## Le premier étage : S'échapper de la salle de bain

Votre cauchemar commence dans la **salle de bain du premier étage**. Contrairement au départ standard à la piscine, vous êtes enfermé. Vous devez trouver la poignée de porte à l'intérieur d'une mallette noire et verte située près de la baignoire. Fixez la poignée à la porte pour sortir dans le couloir.

### L'énigme des canards et la grille d'escalier
Une fois dans le couloir, vous rencontrerez une grille verrouillée par un code à quatre chiffres. Pour résoudre cela, rendez-vous dans la **première chambre à l'étage**. Vous remarquerez des paniers colorés (Blanc, Rouge, Bleu, Vert) et des canards éparpillés partout. Vous devez compter les canards de chaque couleur pour former le code.

| Couleur du canard | Nombre / Chiffre | Logique |
| :--- | :--- | :--- |
| **Blanc** | 7 | Trouvés sur les étagères, les sols et dans le coffre-fort |
| **Rouge** | 5 | Trouvés sur le bureau, le cadre de lit et le coffre-fort |
| **Bleu** | 3 | Trouvés entre les poufs et près de la guitare |
| **Vert** | 4 | Trouvés sur le bureau et près du meuble TV |

**Le code des canards est 7534.** Utilisez-le pour ouvrir la première grille. Cependant, une seconde grille plus bas dans les escaliers nécessite un code différent : **2501**. Ces chiffres se trouvent en observant les tirets jaunes sur les murs de la zone restreinte et de l'allée.

## Rez-de-chaussée : La salle de surveillance et le garage

Après avoir descendu les escaliers, vous trouverez le hall principal barricadé. Vous devez entrer dans la **salle de surveillance** pour progresser. Cette pièce contient une télévision affichant un symbole de **diamant vert** — notez-le pour l'énigme finale.

### L'énigme égyptienne
Dans une petite pièce derrière le bureau de surveillance, vous trouverez trois statues : Néfertiti, Horus et un Pharaon assis. Il s'agit d'une énigme mathématique où vous devez déterminer la valeur de chaque figurine :
*   **Tête de Néfertiti :** 3
*   **Horus :** 4
*   **Pharaon assis :** 12

Entrez le code **1234** sur le clavier. Le mur pivotera pour révéler la **Clé de puzzle 03**, qui déverrouille les portes vitrées menant à l'allée.

### Les clés du garage
Dirigez-vous vers l'allée et entrez dans le garage partiellement ouvert. Sur le mur, des clés plates sont disposées pour former des chiffres. En interagissant avec elles, vous pouvez déchiffrer le code **7048**. Utilisez-le pour déverrouiller l'atelier à l'intérieur du garage afin de récupérer l'une des trois **statuettes d'éléphants** nécessaires pour accéder au sous-sol.

## Les défis du sous-sol et de la cuisine

Pour entrer dans le sous-sol, vous devez placer trois statuettes d'éléphants sur les socles dans la cage d'escalier principale. L'ordre doit être **Blanc, Rouge, puis Noir**. Cela ouvre un passage caché.

### Clavier du sous-sol et preuves
Le sous-sol contient la deuxième pièce de preuve (un attrape-rêves) et une zone grillagée. Pour trouver le code, regardez le tableau à côté de la grille qui représente une fléchette, une boule de billard, de l'argent et un couteau.
1.  **Fléchette :** Regardez la cible de fléchettes à travers les barreaux pour voir le chiffre **9**.
2.  **Boule de billard :** Identifiez le numéro de la boule manquante dans le jeu : **4**.
3.  **Argent :** Ramassez toutes les piles d'argent sur le chariot pour révéler le chiffre **7**.
4.  **Couteau :** Inspectez le magazine "Knife My Wife" et faites-le pivoter pour trouver le **5**.

**Le code du sous-sol est 9475.** L'accès à cette zone vous permet de récupérer la poignée de porte de la cuisine.

### Le coffre de la cuisine et l'énigme des tableaux
Une fois dans la cuisine, comptez les bouteilles sur le comptoir pour ouvrir le coffre-fort sous le bar. Le nombre de bouteilles donne le code **9546**. À l'intérieur se trouve une clé pour la vitrine.

Vous devez ensuite trouver trois tableaux percés de trous et les placer sur le mur numéroté de la cuisine. L'orientation des trous révèle le code pour le salon : **1408**.

| Emplacement du tableau | Position du trou | Chiffre résultant |
| :--- | :--- | :--- |
| **Couloir du hall** | En haut à gauche | 1 |
| **Près de la poterie** | Deux trous | 4 |
| **Vitrine** | En bas au centre | 8 |

## Le salon et l'énigme des symboles TV

Le salon est le dernier obstacle intérieur de la mission **crime scene cleaner nightmare mode affair with death**. La pièce contient six téléviseurs qui doivent afficher des symboles colorés spécifiques pour déverrouiller la sortie vers la piscine. Tout au long du niveau, vous auriez dû noter ces symboles sur divers écrans de télévision.

**La séquence TV correcte :**
1.  **Étoile orange** (Trouvée dans la chambre du premier étage)
2.  **Cœur rose** (Trouvé dans la deuxième chambre)
3.  **Diamant vert** (Trouvé dans la salle de surveillance)
4.  **Croix blanche** (Trouvée dans le garage)
5.  **Cercle bleu** (Trouvé dans le sous-sol)
6.  **Triangle rose** (Trouvé dans la buanderie secrète derrière le livre "Rome Empire")

> **⚠️ Attention :** Interagir avec le livre "Rome Empire" sur l'étagère ouvre une pièce secrète. Si vous entrez et appuyez sur le bouton rouge, la porte peut se refermer, assurez-vous donc d'avoir fini de nettoyer le salon avant d'explorer le compartiment secret.

## La zone de la piscine : Objet de collection Disque Ambré

La zone finale est la piscine. C'est là que le corps du balcon du premier étage finit par atterrir. Après avoir nettoyé le site de l'impact, vous devez chercher le **Disque de musique ambré**.

Pour obtenir le disque, vous devez collecter **16 cubes de puzzle** éparpillés autour de la piscine et du terrain de basket.
*   Deux cubes sont immergés dans la piscine ; utilisez le boîtier électrique à l'extrémité pour vider l'eau.
*   D'autres cubes sont cachés sur de hauts rebords près des plantes et derrière le panier de basket.
*   Un cube est dissimulé dans un coin de la zone secrète qui servait de "Spot Secret" en mode standard.

Une fois que vous avez les 16, disposez-les sur le mur dans la zone de basket pour former un canard jaune. Une porte dérobée s'ouvrira, révélant le disque. ## FAQ

**Q : Où se trouve le corps dans le mode Cauchemar d'Affair with Death ?**

R : Il n'y a qu'un seul corps. Il se trouve initialement dans la chambre du premier étage. Lorsque vous ouvrez les portes du balcon, le corps tombe dans la zone de la piscine en dessous. Vous devez descendre pour le récupérer et l'emmener au camion.

**Q : Je ne trouve pas la troisième statuette d'éléphant. Où est-elle ?**

R : Les trois statues se trouvent : 1) Sous l'escalier, 2) Dans la salle de surveillance, et 3) À l'intérieur de l'atelier verrouillé dans le garage (nécessite le code des clés plates 7048).

**Q : Comment ouvrir le coffre-fort dans la chambre aux canards ?**

R : Interagissez avec le coffre-fort pour entendre une séquence de sons. Cliquez sur les boutons de canards au mur pour imiter la mélodie. Cela déverrouillera le coffre, vous permettant de compter les derniers canards nécessaires pour le code de la grille.

**Q : Quel est le code pour la porte finale du salon dans Crime Scene Cleaner Nightmare Mode Affair with Death ?**

R : Le code est 1408. Il est obtenu en plaçant les trois tableaux "à trous" sur le mur numéroté de la cuisine et en lisant les chiffres visibles à travers les découpes.
